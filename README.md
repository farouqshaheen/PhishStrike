PyPhisher Version 2.0
PyPhisher is a lightweight, portable, easy to use, Python Phishing campaign generator / general mass emailer. For usage information simply enter "Python PyPhisher.py" into a command line.

Features
Send multiple emails using different domains, web host paths, bodies, and campaign identifiers
Replace in-body commands (i.e. {{To_FirstName}}) with their variable counterparts
Create, attach, and track 1x1 pixel tracking gifs
Send both plaintext and html emails with and without attachments.
Body Commands
{{To_FirstName}} - first name of reveiver
{{To_LastName}} - last name of receiver
{{To_Title}} - title of receiver (i.e. mr., mrs., etc.)
{{ToAddress}} - receiver email address
{{From_Name}} - name of sender
{{From_Address}} - sender email address
